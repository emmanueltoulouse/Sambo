# Test de débogage

Ceci est un fichier de test pour reproduire le problème des widgets qui disparaissent lors du chargement d'un document Markdown.

## Contenu du test

- **Gras** et *italique*
- Une liste à puces
- Un `code inline`

```vala
// Un bloc de code
public void test() {
    print("Hello World");
}
```

> Une citation pour tester le rendu

Fin du test.
